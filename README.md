# Adaptive Chips
## Компонент отображающий чипсы и popover с ними
## Описание работы

### Используемый стек:
- TypeScript;
- React v.19;
- Vite;

### Линтинг:
- Eslint;
- Prettier;

### Дополнительные библиотеки:
- не используются;

## Функуционал проекта:
- Отображение чипсов в одну строку;
- Если чипс не помещается по ширине контейнера, то он переносится в сплывающий popover, который становится доступен по клику на специальную кнопку;
- Все чипсы доступны к выбору по клику;
- Компонент адаприуется к ширине экрана;
- Все компоненты являются переиспользуемыми;

### Реализация:
- Popover (popup) реализован с помощью нативного Popover API без использования сторонних библиотек;

## Установка и запуск
### 1. Скачать или склонировать репозиторий:

```sh
git clone https://github.com/AlexAven/sirius-task.git
```

### 2. Установить зависимости:

```sh
cd sirius-task
npm install
```

### 3. Запустить приложение локально:

```sh
npm run dev
```
